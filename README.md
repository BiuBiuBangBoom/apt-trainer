# APT Trainer

行政职业能力测验（行测）速算练习工具，交互式终端应用。

## 运行

```bash
conda activate python313
python main.py            # 正常运行
python main.py -n 20      # 限制 20 题
python main.py -t 30      # 每题限时 30 秒

python -m unittest test_all -v   # 运行测试
```

无外部依赖，仅需 Python 3.13 标准库。

## 题型（10 种）

| # | 题型 | 示例 |
|---|------|------|
| 1 | 两位数乘一位数 | 26 × 8 |
| 2 | 一位数加一位数 | 7 + 5 |
| 3 | 一位数乘一位数 | 6 × 9 |
| 4 | 三位数除以两位数 | 456 ÷ 38 |
| 5 | 分数比较 | 3/7 与 5/12 |
| 6 | 最近百化分 | 12.5% → 1/8 |
| 7 | 三位数乘一位数 | 345 × 7 |
| 8 | 现期增长率估算增长量 | 基期 1234，增长 5.6% |
| 9 | 两位数减一位数 | 83 − 7 |
| 10 | 幂次数 | 2⁸ |

## 练习模式（3 种）

- **跑图模式** — 每题显示答案，无正确性判断
- **跑测模式** — 每题判断对错，退出时显示统计
- **考试模式** — 全程无反馈，退出时显示完整统计

## 架构

采用双层策略模式：

- **第一层** — 执行模式（`ExamStrategy`）：控制答题流程和反馈方式
- **第二层** — 计算题型（`QuestionType`）：只负责生成题目和验证答案

`QuestionType.generate_question()` 是纯函数，返回自包含的 `Question` dataclass（题目文本 + 答案 + 可选元数据），无副作用。`Session` 管理题目生成和答题记录。
